## 🎓 AI-Powered Data Analysis: Student Engagement & Churn Prediction

### Project Overview

This project was completed during a 4-week **AI Data Analyst Remote Internship** supported by Rochester Institute of Technology's (RIT) Tiger STRIPES Program and powered by Excelerate. The primary goal was to analyze student engagement data to identify patterns leading to drop-offs and build a predictive model to improve learner retention.

### ⭐ Key Achievements

- **Award:** Awarded Star Performer for outstanding achievement during the internship
- **Data Scope:** Cleaned and analyzed **8,263 records** across **27 features**
- **Accuracy:** Developed a Random Forest Classifier that outperformed other models (Logistic Regression and Decision Tree) in predicting student churn with high precision

### 🔍 The Problem

Excelerate needed to understand the "Learner Funnel"—specifically, why students sign up but fail to reach "Rewarded" status. The project focused on identifying friction points and predicting which learners were at risk of dropping out.

### 📊 Project Workflow

**Week 1: Data Cleaning & Feature Engineering**
- Standardized 8,558 raw rows
- Resolved 3,804 missing values
- Created new features (Signup Entry Lag, Engagement Duration)

**Week 2: Exploratory Data Analysis (EDA)**
- Identified trends in signups and completion rates

**Week 3: Churn Analysis**
- Built predictive models to identify factors contributing to student drop-offs

**Week 4: Final Deliverables**
- Comprehensive report, Power BI dashboard, and prototype recommendation system

### 💡 Key Insights & Recommendations

**Strongest Churn Predictor:** "Signup Entry Lag"
- Learners who experienced an administrative delay of more than 100 days were significantly more likely to drop out

**Engagement Insight:**
- Faster application after signing up correlates with higher success and commitment

**Recommended Strategy:**
- Implement a "Quick Apply" flow
- Set up early risk alerts to trigger advisor outreach for high-risk learners
